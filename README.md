# In Falsus Demo - Asset Decryptor (v2.1)

本工具用于全自动解密《In Falsus Demo》游戏中的加密资源文件，并将其**自动重命名及分类**。

**核心功能：**
1.  **自动解密**：将加密的二进制文件转换为可播放的音频 (.ogg) 和可读文本 (.txt)。
2.  **自动还原**：基于映射表将乱码文件名恢复为原始文件名（如 `bgm/MainTitle.ogg`）。
> **注意**：解密后的文件仅供个人学习、研究或 Mod 制作使用。

##  准备工作

在开始之前，请确保您已经：
1.  下载并安装了《In Falsus Demo》。
2.  找到了游戏的安装目录（通常在 Steam 库 -> 右键游戏 -> 管理 -> 浏览本地文件）。
3.  准备好tool_v2.1.zip或您自己编译完后的程序。

---

##  使用步骤

### 1. 部署工具
1.  解压tool_v2.1.zip，将 **`SamDecryptor.exe`** 和 **`StreamingAssetsMapping.json`** （文件名映射表）放入游戏根目录（即 `if-app.exe` 所在的目录）。

### 2. 准备加密源文件 (可选)
工具默认会自动读取游戏自带的 `sam` 文件夹。
为了安全起见，您也可以手动操作：
1.  在游戏根目录新建文件夹 **`EncryptedSource`**。
2.  将 `..\if-app_Data\StreamingAssets\sam\` 中的所有文件复制进去。
*(如果存在 `EncryptedSource`，工具会优先读取它，避免修改游戏原文件)*

### 3. 运行解密
双击运行 **`SamDecryptor.exe`** ，并等待控制台窗口出现。

### 4. 获取结果
游戏根目录，您会看到：  
**`RestoredAssets`**： 存放已重命名并分类好的最终文件。
同时，根目录下会生成 **`decrypt_log.txt`**，其中记录了每个文件的恢复记录。

---

## 常见问题排查

若未成功生成文件：
1.  **检查日志**：查看 `decrypt_log.txt`。
2.  **检查路径**：确保 `sam` 文件夹存在，或者您正确创建了 `EncryptedSource`。
3.  **检查权限**：尝试以管理员身份运行此程序。
